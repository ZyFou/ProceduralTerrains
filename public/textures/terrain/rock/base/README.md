# Rock — base

Default texture set for the Rock material:

- `rock_diffuse.jpg`
- `rock_displacement.exr`
- `rock_normal_dx.jpg`
- `rock_roughness.jpg`
- `rock_ao.jpg`

Empty for now — drop matching files in and this becomes the material's out-of-the-box look. Add more look-alikes via the "Add Variant" button in Surface > Textures; each gets its own folder here (e.g. `variant_1/`) with the same filenames.
